This chapter details the interface routines for the Hierarchical Data Format. General notes about the interface are presented below, followed by individual descriptions of the routines.

    NOTE: The routines HDF_BROWSER and HDF_READ , introduced in IDL version 5.1, allow you to read HDF data files and import data into IDL using a graphical user interface. Using these two routines, you can avoid the need to use most of the rest of IDL's HDF interface.

This version of IDL supports HDF 4.0. However, features new to HDF 4.0 are not supported in this version of IDL. This features include, but are not limited to:

  • The new Multi-file Annotation Interface (ANxxx).
  • The new Multi-file Raster Image Interface (GRxxx).
  • HDF reading of CDF files.
  • Reading/Writing of compressed HDF scientific datasets (SDS).
  • Reading/Writing of SDS using the new n-bit interface.
