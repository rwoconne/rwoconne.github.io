HDF_DFSD_READREF

The HDF_DFSD_READREF procedure specifies the reference number of the Hierarchical Data Format file to be read by the next call to HDF_DFSD_GETINFO or HDF_DFSD_GETDATA.

Calling Sequence

HDF_DFSD_READREF, Filename, Refno

Arguments

Filename

A scalar string containing the name of the file to be read.

Refno

The reference number of the desired SDS.
