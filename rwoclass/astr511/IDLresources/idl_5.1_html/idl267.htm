HDF_DELDD

The HDF_DELDD procedure deletes a tag or reference from the list of data descriptors in a Hierarchical Data Format file.

Calling Sequence

HDF_DELDD, FileHandle, Tag, Ref

Arguments

FileHandle

The HDF file handle returned from a previous call to HDF_OPEN.

Tag

The data descriptor tag to delete.

Ref

The data descriptor reference number to delete.
