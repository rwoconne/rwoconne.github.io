HDF_DFP_NPALS

The HDF_DFP_NPALS function returns the number of palettes present in a Hierarchical Data Format file. This number includes palettes associated with RIS8 (8-bit raster) images.

Calling Sequence

Result = HDF_DFP_NPALS( Filename )

Arguments

Filename

A scalar string containing the name of the desired HDF file.
