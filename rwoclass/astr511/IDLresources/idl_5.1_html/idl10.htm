BESELI

The BESELI function returns the I Bessel function of order N for the argument X . If X is double-precision, the result is double precision, otherwise the argument is converted to floating-point and the result is floating-point.

Calling Sequence

Result = BESELI( X, N )

Arguments

X

The expression for which the I Bessel function is required. The result will have the same dimensions as X .

N

The order of the I Bessel function to calculate. N must be an integer greater than or equal to 0.
