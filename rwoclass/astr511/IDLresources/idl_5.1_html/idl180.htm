REWIND

The REWIND procedure rewinds the tape on the designated IDL tape unit. REWIND is available only under VMS. See the description of the magnetic tape routines in VMS-Specific Information .

Calling Sequence

REWIND, Unit

Arguments

Unit

The magnetic tape unit to rewind. Unit must be a number between 0 and 9, and should not be confused with standard file Logical Unit Numbers (LUNs).
