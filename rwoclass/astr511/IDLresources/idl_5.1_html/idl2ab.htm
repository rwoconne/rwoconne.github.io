HDF_SD_ISCOORDVAR

The HDF_SD_ISCOORDVAR function returns True (1) if the supplied dataset ID represents a NetCDF "coordinate" variable. Otherwise, False (0) is returned.

Calling Sequence

Result = HDF_SD_ISCOORDVAR( SDS_ID )

Arguments

SDS_ID

An SD dataset ID as returned by HDF_SD _SELECT or HDF_SD_CREATE.
